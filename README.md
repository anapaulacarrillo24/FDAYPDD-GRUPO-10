# FDAYPDD-GRUPO-10
Fundamentos de análisis y procesamiento de datos para TFM

OBJETIVO. 
Implementar un proceso de extracción y análisis exploratorio de datos a partir de fuentes web institucionales, utilizando técnicas de web scraping y procesamiento de datos, con el propósito de obtener información relevante sobre la estructura multicampus de una institución universitaria y apoyar el análisis preliminar para el diseño de un sistema WMS aplicado a la gestión de activos fijos.

METODOS.
Se emplearon técnicas de obtención y análisis de datos a partir de fuentes públicas, utilizando web scraping con Python para extraer información institucional de una universidad multicampus. Los datos obtenidos fueron procesados y estructurados para realizar un análisis exploratorio que permitió contextualizar la complejidad operativa de la institución y sustentar el diseño de un sistema de gestión de almacenes (WMS) en el marco del Trabajo Fin de Máster. 

CONCLUSIONES.
La aplicación de técnicas de obtención y procesamiento de datos a partir de fuentes web permitió estructurar información institucional no estructurada y realizar un análisis exploratorio que evidenció la distribución multicampus de la institución y su complejidad operativa, proporcionando una base analítica para justificar el diseño de un sistema de gestión de almacenes (WMS) orientado a la trazabilidad y control de activos fijos en el contexto del Trabajo Fin de Máster.
